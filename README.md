# SONAR Rock vs Mine Prediction using Machine Learning

This project predicts whether an object is a **Rock** or a **Mine** using the SONAR dataset and **Logistic Regression**.

## Technologies Used
- Python
- NumPy
- Pandas
- Scikit-learn

## Algorithm
- Logistic Regression

## Dataset
- SONAR Dataset (208 instances, 60 features)
- Label: Rock (R), Mine (M)

## How to Run
1. Upload `sonar.csv`
2. Run all cells in the notebook

## Output
- Training Accuracy ~ 83%
- Testing Accuracy ~ 76%
